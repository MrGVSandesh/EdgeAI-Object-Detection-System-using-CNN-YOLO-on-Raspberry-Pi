🚀 EdgeAI Object Detection System using CNN (YOLO) on Raspberry Pi
🧠 Project Overview

This project implements a real-time object detection system using Deep Learning (CNN + YOLO architecture) deployed on a Raspberry Pi edge device. It captures live video input and performs on-device inference to detect and classify multiple objects with bounding boxes and confidence scores.

The system is optimized for low-latency edge computing, making it suitable for real-world AI applications such as surveillance, smart monitoring, and IoT-based vision systems.
